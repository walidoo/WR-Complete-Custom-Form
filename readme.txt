Is a Wordpress Plugin has a shortcode for signup and login form for wordpress websites.

=== Description ===

Features:

1. Add a new page for register page and add shortcode for register form with a validation options
for invalid data when user registers a new account
2. Add a new page for login page and add shortcode for login form with a validation option 
for invalid account


=== Installation ===
1.  Upload your plugin folder to the '/wp-content/plugins/' directory.
2.  Activate the plugin through the 'Plugins/' menu in WordPress.
3.  Add a new page for register form and add short code [cr_custom_registration]
4.  Add a new page for login form and add short code [cr_custom_login]
